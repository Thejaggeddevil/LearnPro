##  What LearnPro Offers

-  Course browsing with categorized content
-  Detailed course pages with structured lessons
-  User-friendly interface focused on learning flow

## Features

- Course listing and navigation
- Clean UI for distraction-free learning
- Modular screen-based architecture
- Scalable design for future features (auth, progress tracking, etc.)

## Tech Stack

- Frontend: Kotlin, Jetpack Compose  
- State Management: ViewModel  
- UI: Material Design  
- Architecture: Component-based / MVVM

## Project Status

LearnPro is a work-in-progress project focused on building a strong UI and scalable foundation for a full learning platform.
